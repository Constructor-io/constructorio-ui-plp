## Props
<Controls />

### `PlpSearchResponse`
  | property              | type                     | description|
  | :---------------------| -------------------------| :--------- |
  | resultId              | `string`                 | The result ID' |
  | totalNumResults       | `number`                 | The total number of results across all pages |
  | numResultsPerPage     | `number`                 | The number of results per page |
  | results               | `Array<Item>`            | Array of Constructor result items |
  | facets                | `Array<Facet>`           | Array of Constructor filters  |
  | groups                | `Array<Group>`           | Array of group items the results belongs to  |
  | sortOptions           | `Record<string, any>[]`  | Array of available sort options  |

**Or** 

### `PlpSearchRedirectResponse`
  | property              | type                     | description|
  | :---------------------| -------------------------| :--------- |
  | resultId              | `string`                 | The result ID' |
  | redirect              | `Partial<Redirect>`      | The redirect object containing redirect url |
