## `Sort`
The Sort component is a dynamic component.
It is responsible for displaying the sort options based on user's search or browse response. 

**Dynamic Options Rendering:**
- Automatically updates and displays sort options based on user requests.

### Basic Example
```jsx
  import React from 'react';
  import { CioPlp, Sort } from '@constructor-io/constructorio-ui-plp';

  const apiKey = 'MY_API_KEY';

  function MyComponent() {

    return (
      <CioPlp apiKey={MY_API_KEY}>
        <Sort />
      </CioPlp>
    );
  }
```

### Using RenderProps
If you prefer to handle the rendering of the sort yourself, you may pass a Render Props function to the component.

```jsx
import { CioPlp, Sort } from '@constructor-io/constructorio-ui-plp'

const apiKey = 'MY_API_KEY';

function myApp() {
  ...

  return (
    <CioPlp apiKey={MY_API_KEY}>
      <Sort />
        {({sortOptions, selectedSort, changeSelectedSort}) => sortOptions.map((option) => <li>{option.displayName}</li>)}
      </Sort>
    </CioPlp>
  )
}
```
