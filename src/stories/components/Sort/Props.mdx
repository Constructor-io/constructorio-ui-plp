## Props
<ArgTypes />

### PlpSortOption
  | property              | type                   | description       | 
  | :-----------------| :--------------------------| :---------------- |
  | sortBy            | `string`                   | Name of the sort option |
  | sortOrder         | `ascending \| descending`  | Order the results will be sorted in |
  | displayName       | `string`                   | Name of the sort option as it is displayed to end users |
  | status            | `selected \| ''`           | "selected" if this sort option is applied, empty string otherwise |
