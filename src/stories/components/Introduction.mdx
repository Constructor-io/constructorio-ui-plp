# Using Components
Choose to integrate simply using the component [CioPlp](./?path=/docs/components-cioplp--code-examples) that takes in a set of configuration options.

Alternatively, utilize the suite of sub-components* that make up the [CioPlp](./?path=/docs/components-cioplp--code-examples) and create your own PLP layout. 
<div>
  <span className="badge note">Note</span>*These sub-components must be nested under the CioPlp component to access the required context.
</div>
