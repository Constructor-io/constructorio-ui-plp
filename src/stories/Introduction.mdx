# Welcome to Constructor's PLP UI

Integration with Constructor's APIs has never been easier with **comprehensive components** that take care of both the tracking and results retrieval, 
allowing you to focus on developing your own unique site experience.

If you prefer hooks, we provide the **useCioPlp** hook that accepts a suite of configuration parameters and returns all the underlying logic that powers all our components.
